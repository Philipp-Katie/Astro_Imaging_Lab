The code for the Astro Imaging Lab written by Katherine Whitby and Philipp Sonnenschein 

Written for Python 3.8.

The folder contains 3 files:
1. Main_code
2. Code_validation
3. Synth_data_validation


Files should be progressively executed in the same order as listed above.

Comments at the top of each cell provide a short overview of the task being executed.

Additional validation is provided in the Synth_data_validation with synthesised galaxies.
